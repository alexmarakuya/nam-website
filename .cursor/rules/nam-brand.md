# NāM Brand Context

This file provides automatic context for any AI session working on NāM projects. For full source material, see `/NAM-brand-reference.md` in the project root.

## What NāM Is

NāM is a long-term, place-based creative campus on Koh Phangan, Thailand. A co-creation space — not co-working, not co-living. It's for visionaries, creators, builders, and change-makers who want to do real, meaningful, self-directed work. The brand archetype is "a school of magic" — Magician meets Fool.

## Core Thesis

The co-creation space is the third place for the post-AI world. AI is automating routine work; what's left is human creation. NāM is the space designed for that reality — where people come together to build what matters, not in silos but in community.

## Voice & Tone

**The Magician meets the Fool — in boots, not robes.** Clear over clever. Grounded wisdom. Depth with fire. Encouragement, not hype.

**Words to love**: build, rhythm, ritual, real work, alignment, play, clarity, curiosity, magic, focus, precision, flow, depth, practice, experiment, momentum, grounded, support, community, dare, brave, unique, yourself.

**Words to avoid**: hustle, grind, optimize, scale fast, guru, manifest, high-vibe, productivity hacks, exclusive, premium, retreat, escape, luxury, fluff.

## Key Principles

1. Place First — the land is an active participant
2. Time as Design Material — rhythm over acceleration
3. Depth Over Throughput — quality of attention over quantity
4. Care Is Infrastructure — wellbeing is a condition created together
5. Learning Through Proximity — people learn by being near each other
6. Autonomy With Belonging — freedom strengthened by context
7. Soft Thresholds — open, but not anonymous
8. Visibility of Systems — infrastructure teaches
9. Designed for Change — the space is never finished
10. Stewardship Over Ownership — held, not consumed
11. Play and Experimentation — curiosity keeps systems alive

## Build Phases

- Phase I: Work/studio building, Café, Outdoor areas, Water/energy systems
- Phase II: Accommodation, Shared living, Orientation
- Phase III: Carpentry, Pottery, Maker spaces
- Phase IV: Additional studios, Events, Incubation

## Island Context

- Water crisis (harvesting/storage as case study)
- Power outages (solar for uptime and resilience)
- Plastic waste crisis (composting, reusables, packaging reduction)

## Icon Library

We use **Lucide** (https://lucide.dev) for all UI icons across the site. Icons are inlined as SVGs, not loaded via CDN or JS bundle.

Standard attributes for inline Lucide icons:
```
viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"
```

Override `stroke` color to match context (e.g. `#f5f2e9` on dark/colored backgrounds).

Icons currently in use:
- `user` — member profile / publish your profile
- `megaphone` — announcements / project updates
- `hammer` — building / shaping the space

When adding new icons, always source from Lucide and document them here.
